#define USER_DEBUG_VPRINTF   vprintf

Debug output from the IIK, Writes a formatted string to stdout, expanding the format tags with the value of the argument list arg. This function behaves exactly as printf except that the variable argument list is passed as a va_list instead of a succession of arguments.

In the C library the prototype for vprintf is defined as vprintf(const char *format, va_list ap);

#define ASSERT (   cond)    assert(cond)
