Operating System Callbacks

The IIK interfaces to the platform's OS through the callbacks listed below. These are called through the application callback described in the Public API Overview. The class id is idigi_class_operating_system.

  1. malloc
  2. free
  3. System Uptime
  4. Sleep

malloc

Callback is used to dynamically allocate memory.

This callback is trapped in application.c, in the Sample section of Public Application Framework and implemented in the Platform function app_os_malloc() in os.c.

See also:
app_os_malloc()
app_os_free()
idigi_os_free
Arguments
Name Description
class_id idigi_class_operating_system
request_id idigi_os_malloc
request_data Pointer to number of bytes to be allocated
request_length The size of *request_data which is sizeof size_t.
response_data Returns a pointer to memory for allocated address
response_length N/A
Return Values
Values Description
idigi_callback_continue Callback successfully allocated memory
idigi_callback_abort Callback was unable to allocate memory and callback aborts IIK
idigi_callback_busy Memory is not available at this time and needs to be called back again

Example:

 idigi_callback_status_t idigi_callback(idigi_class_t const class_id, idigi_request_t const request_id
                              void * const request_data, size_t const request_length,
                              void * response_data, size_t * const response_length)
 {

     if (class_id = idigi_class_operating_system && request_id.os_request == idigi_os_malloc)
     {
         size_t size = *(size_t *)request_data;
         void ** ptr = (void **)response_data;

         *ptr = malloc(size);
         if (*ptr == NULL)
         {
             return idigi_callback_abort;
         }
     }
     return idigi_callback_continue;
 }

free

Callback is called to free previously allocated memory.

This callback is trapped in application.c, in the Sample section of Public Application Framework and implemented in the Platform function app_os_free() in os.c.

See also:
app_os_free()
app_os_malloc()
idigi_os_malloc
Arguments
Name Description
class_id idigi_class_operating_system
request_id idigi_os_free
request_data Pointer address to be freed
request_length N/A
response_data N/A
response_length N/A
Return Values
Values Description
None None

Example:

 idigi_callback_status_t idigi_callback(idigi_class_t const class_id, idigi_request_t const request_id
                              void * const request_data, size_t const request_length,
                              void * response_data, size_t * const response_length)
 {

     if (class_id = idigi_class_operating_system && request_id.os_request == idigi_os_free)
     {
         free((void *)request_data);
     }
     return idigi_callback_continue;
 }

System Uptime

This callback is called to return system up time in seconds. It is the time that a device has been up and running.

This callback is trapped in application.c, in the Sample section of Public Application Framework and implemented in the Platform function app_os_get_system_time() in os.c.

See also:
app_os_get_system_time()
Arguments
Name Description
class_id idigi_class_operating_system
request_id idigi_os_system_up_time
request_data N/A
request_length N/A
response_data Pointer to unsigned long integer memory where callback writes the system up time to (in seconds)
response_length N/A
Return Values
Values Description
idigi_callback_continue Callback successfully returned the system time
idigi_callback_abort Error occurred and callback aborted IIK

Example:

 idigi_callback_status_t idigi_callback(idigi_class_t const class_id, idigi_request_t const request_id
                              void * const request_data, size_t const request_length,
                              void * response_data, size_t * const response_length)
 {

     if (class_id = idigi_class_operating_system && request_id.os_request == idigi_os_system_up_time)
     {
         time((time_t *)response_data);
     }
     return idigi_callback_continue;
 }

Sleep

This callback is called to relinquish control in the multi-threaded idigi_run() model.

This callback is trapped in application.c, in the Sample section of Public Application Framework and implemented in the Platform function app_os_sleep() in os.c.

See also:
app_os_sleep()
Multi-threading and the IIK Threading Model
Arguments
Name Description
class_id idigi_class_operating_system
request_id idigi_os_sleep
request_data Number of seconds to sleep up to (the user can sleep less)
request_length The size of request_data which is unsigned int
response_data N/A
response_length N/A
Return Values
Values Description
idigi_callback_continue Callback successfully was able to sleep
idigi_callback_abort Error occurred and callback aborted IIK

Example:

 idigi_callback_status_t idigi_callback(idigi_class_t const class_id, idigi_request_t const request_id
                              void * const request_data, size_t const request_length,
                              void * response_data, size_t * const response_length)
 {

     if (class_id = idigi_class_operating_system && request_id.os_request == idigi_os_sleep)
     {
         unsigned int *timeout_seconds = request_data;
         usleep(*timeout_seconds * MICROSECONDS_PER_SECOND);
     }
     return idigi_callback_continue;
 }
