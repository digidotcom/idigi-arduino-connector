Detailed Description

Data service block structure is used to send data between iDigi server and client. This structure is used in idigi_data_service_put_request and idigi_data_service_device_request callbacks.

When message type is idigi_data_service_type_need_data in idigi_data_service_msg_request_t, callback needs to update this structure with data to be sent to server.

When message type is idigi_data_service_type_have_data in idigi_data_service_msg_request_t, this structure contains data for callback to process.
