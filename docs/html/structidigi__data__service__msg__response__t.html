Detailed Description

Data service message response structure is used in idigi_data_service_put_request and idigi_data_service_device_request callbacks to return data to be sent to iDigi server, or to cancel the message.

Field Documentation

void* user_context

Used for idigi_data_service_device_request callback's context which will be returned on subsequent callbacks for its reference. For idigi_data_service_put_request callback, it's not used.

idigi_msg_error_t message_status

Callback writes error status when it encounters error and cancels the message

idigi_data_service_block_t* client_data

Pointer to memory where callback writes data to for idigi_data_service_type_need_data message type
