iDigi Web Services

Applications can communicate to devices over the iDigi Cloud using iDigi Web Services, requests are sent to the iDigi server using Server Command Interface (SCI). The IIK supports a subset of the SCI operations supported by iDigi, the supported SCI operations are listed in the table below:

Name Description
update_firmware Updates the firmware of the device
disconnect Sends a request to the device to disconnect from the server
query_firmware_targets Gets a list of firmware targets on the device
data_service Sends messages to devices over the data service

To learn more on how to use iDigi Web Services read this document, refer to Chapter 5. SCI (Server Command Interface).
