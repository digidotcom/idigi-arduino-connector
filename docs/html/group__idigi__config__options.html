Define Documentation

#define IDIGI_LITTLE_ENDIAN

When defined, the system is little endian.

Note:
For Big endian systems, this line should be commented out.
#define IDIGI_DEBUG

When defined, IIK private library includes debug support. IIK state and debug data will be displayed using the USER_DEBUG_VPRINTF tags. Macro error testing is compiled and implemented using the ASSERT macro.

This debug feature can be eliminated in a product release, which will greatly reduce memory codespace and RAM needs, by changing this line in idigi_config.h:

 #define IDIGI_DEBUG

To this: 

 //#define IDIGI_DEBUG

#define IDIGI_FIRMWARE_SERVICE

When defined, IIK private library includes the Firmware Download Service.

To disable the Firmware Download Service feature, change this line in idigi_config.h:

 #define IDIGI_FIRMWARE_SERVICE

To this: 

 //#define IDIGI_FIRMWARE_SERVICE

Firmware Access Support
#define IDIGI_NO_COMPRESSION

This is a compilation convenience definition. See IDIGI_COMPRESSION.

#define IDIGI_COMPRESSION

When defined, the IIK includes the compression support used with the Data Service.

By default, compression support is disabled. To enable Optional Data Compression Support, change this line in idigi_config.h: By default, optional compression support is disabled.

To enable compression, change the following from idigi_config.h:

 #define IDIGI_NO_COMPRESSION
 #if (!defined (IDIGI_NO_COMPRESSION) || defined(DIGI_REQUIRED_TO_GENERATE_INTERNAL_DOCUMENATION))

 ...

 #define IDIGI_COMPRESSION
 #endif

To this: 

 //#define IDIGI_NO_COMPRESSION
 #if (!defined (IDIGI_NO_COMPRESSION) || defined(DIGI_REQUIRED_TO_GENERATE_INTERNAL_DOCUMENATION))

 ...

 #define IDIGI_COMPRESSION
 #endif

Or just this: 

 #define IDIGI_COMPRESSION
Note:
When included, this requires the zlib library.
