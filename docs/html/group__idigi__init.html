idigi_handle_t idigi_init ( idigi_callback_t const  callback)

This API is called once upon startup to allocate and initialize the IIK. It takes the application-defined callback function as an argument; this callback is used by the IIK to communicate with the application. This function must be called before all other IIK APIs. The idigi_callback_t is defined below.

Parameters:
[in]callbackCallback function that is used to interface between the application and the IIK.
Return values:
NULLError was found and the IIK is unable to initialize.
Not NULLHandle used for subsequent IIK calls.

Example Usage: 

    idigi_handle = idigi_init(application_callback);
